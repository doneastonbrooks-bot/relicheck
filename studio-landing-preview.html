The pattern across all four studios:
  • Topnav stays platform-neutral; the studio identity shows up as the active dropdown tint, the blob, the headline color, the tile hover, the SVG illustration
  • Asymmetric brand blob top-left carries the studio icon — punches identity before you read a word
  • Headline uses Fraunces (your marketing serif), set in the studio's accent. Sub-headline navy. Body neutral.
  • Single primary CTA: the most-common entry action for that studio
  • Four quick-action tiles below: the second-most-common entry actions, hover lifts and tints to the accent
  • Live-stat strip at the bottom: 3 numbers that prove the studio is alive (or honest "EARLY" / "BETA" status if not yet built out)
  • Subtle entrance animations: eyebrow fades in, headline + sub rise, CTA rises last. Illustration floats gently on a 5s loop.
  • Watermark letters scatter in the background (faded, accent-tinted) — echoes the reference screenshot's ghost letter texture
