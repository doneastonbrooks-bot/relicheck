Connect a device

The ReliCheck MM Studio desktop app is asking to connect to your ReliCheck account. Type the 8-character code shown in the app below to approve the pairing.

Where do I see this code?
Open ReliCheck MM Studio on your computer, click Import from ReliCheck, and a code will appear in a small window. The code expires after 5 minutes. If yours has expired, start a new pairing in the desktop app.
