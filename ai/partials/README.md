# ReliCheck Intelligence tool-card partials

One Markdown file per tool. Each partial holds the body copy for one card so the stage pages and any future use-case page (Education, HR & Teams, Research) can reuse the same source without copy drift.

## Card format

Each file uses the same shape:

```
# Tool Name

**Where it lives:** Short location string.

Body paragraph(s). Voice rules: no em dashes, no personification, no
unintentional anaphora.
```
